Getting Started with Talon

Talon may seem like an intimidating application, but rest assured that it's easy once you start!

To get started, learn What is Talon?

Then, review the following documentation:
  • Setting up Talon on your Computer
  • Learning Talon Voice Commands
  • Adding Customization to Talon

If you need help, join the Talon Slack and ask questions in the Slack's #help channel.

The best way to learn about Talon is to play around with it. Over time, you will become more familiar with the application and use it like a pro. Happy experimenting!
