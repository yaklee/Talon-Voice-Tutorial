Chaosparrot on GitHub has also designed a has designed a Talon practice webpage with seven voice command lessons for basic Talon usage.

  • What is command, dictation, and sleep mode in Talon?
    Modes change how Talon interacts with your speech.
  • What are the voice commands for mode switching?
    You can voice command Talon to switch between three modes: command, dictation, or sleep. By default, Talon will be in command mode.
  • What are the voice commands for dictating text?
    While Talon has a dictation mode, you can also dictate text in command mode.
  • What are the voice commands for punctuation?
    You can use punctuation for writing and coding using voice commands in Talon's command mode.
  • What are the voice commands for web browser tabs?
    You can interact with web browser tabs using Talon in command mode.
  • What are the voice commands to control your mouse?
    You can control your mouse by using Talon in command mode.
  • What are the voice commands for scrolling?
    You can scroll using Talon in command mode.
  • What are the voice commands for interacting with computer applications?
    You can interact with computer applications using Talon in command mode.
