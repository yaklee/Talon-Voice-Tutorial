What are the Talon voice commands for mode switching?

You can voice command Talon to switch between three modes: command, dictation, or sleep. By default, Talon will be in command mode.

Table 1. Mode Switching Commands
CommandFunction
Talon wakeEnable speech recognition
Talon sleepDisable speech recognition
Dictation modeSwitch to dictation mode
Command modeSwitch to command mode
