How do I get started with Talon?

This page will guide you through the setup process, basic usage, and customization options for Talon.

Before you can use Talon with your voice, you must install three components:

  • Talon application
  • Voice command files
  • Speech recognition engine

Once you have installed these components, it is recommended you play around with Talon. Start by learning about differences between the command, dictation, and sleep mode for Talon.

Next, become familiar with voice commanding in Talon. Chaosparrot on GitHub has designed a Talon practice webpage with 7 voice command lessons for basic Talon usage. Additional voice commands for basic Talon usage are also listed below:

  • What are the Talon voice commands for dictating text?
  • What are the Talon voice commands for punctuation?
  • What are the Talon voice commands for web browser tabs?
  • What are the Talon mouse commands?
  • What are the Talon voice commands for scrolling?
  • What are the Talon voice commands for application interaction?
After you start using Talon, you may want to customize it to enhance your experience. It is highly recommended to install the Rango browser extension to improve your webpage interactions. Other common customization include:
  • Editing Talon voice commands
  • Adding Talon-compatible software tools
  • Connecting accessibility hardware tools to Talon (e.g. eye tracker, noise recognition software)

If you need any help, join the Talon Voice Slack and ask questions in the Slack's #help channel.

  • How do I install Talon for macOS?
    You must install Talon to your operating system to run the application. This process should take 5 minutes.
  • How do I download voice commands to Talon for macOS?
    Voice commands that are spoken to Talon allow you to control your computer. This process should take 5 minutes.
  • How do I install a speech recognition engine to Talon for macOS?
    A speech recognition engine allows Talon to recognize your voice commands and respond to them with action. This process should take 5 minutes.
  • How do I check my microphone in Talon?
    The microphone allows your speech to be picked up by Talon's speech recognition engine.
